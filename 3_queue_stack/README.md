3. while 輸流快閃ｓｔａｃｋ　內的值的次數。 然後間隔1秒， 再閃下一個ｓｔａｃｋ內的值
  （也就是用閃燈代替ｄｉｓｐｌａｙ)           (push for stack)
4. 閃完如果按了ｐｕｓｈ　ｂｕｔｔｏｎ,　代表ｐｏｐ　ｓｔａｃｋ。 從sｔａｃｋ　中取值。 (pop for stack)
5. 不斷重覆　３~４, 直到程式結束。
